# 第三轮组合计划使用说明

> 本文档说明如何在 IDEA 终端里使用 06-07 第一版组合计划功能。  
> 当前组合计划只生成买入计划，不生成成交，不连接券商。

---

## 1. 当前能力

第三轮代码支持：

- 批量读取多个 `decision_result`。
- 回读对应 `strategy_snapshot`。
- 执行最小风控检查。
- 过滤不可交易结论。
- 对候选股票透明评分。
- 根据可用现金、现金保留线、单票上限分配资金。
- 输出组合计划报告。

当前不支持：

- 自动实盘下单。
- 复杂行业集中度。
- 波动率优化。
- 历史回放。
- 业务直接写入 SQLite 主账本。

---

## 2. 前置条件

先初始化模拟账户：

```powershell
python .\paper_trading.py init --account paper_default --cash 100000
```

每个交易日先换日：

```powershell
python .\paper_trading.py rollover --account paper_default --trade-date 2026-07-06
```

然后生成多个第一轮买入决策：

```powershell
python .\build_decision.py 002563 --task buy
python .\build_decision.py 600398 --task buy
python .\build_decision.py 601138 --task buy
```

---

## 3. 生成组合计划

```powershell
python .\portfolio_plan.py build --account paper_default --trade-date 2026-07-06 --decision-dir data/decision_results
```

只看指定股票：

```powershell
python .\portfolio_plan.py build --account paper_default --trade-date 2026-07-06 --include 002563,600398
```

限制候选数量：

```powershell
python .\portfolio_plan.py build --account paper_default --trade-date 2026-07-06 --max-candidates 5
```

## 4. 输出文件

```text
data/risk_control/risk_checks.jsonl
data/portfolio_construction/allocation_plans.jsonl
data/portfolio_construction/order_intents.jsonl
data/reports/allocation_report_<account>_<trade_date>.md
```

这些文件是本地运行结果，不建议提交 Git。

---

## 5. 怎么看报告

报告重点看：

- 今日买入预算。
- 哪些股票进入买入计划。
- 每只股票建议买入金额。
- 每只股票建议买入数量。
- 哪些股票被拒绝。
- 哪些股票只记录观察。
- 每只股票的评分拆解。

`READY_FOR_CONFIRM` 表示可进入模拟盘或人工确认。

注意：

```text
组合计划不是成交记录。
```

真正模拟成交仍然需要：

```powershell
python .\paper_trading.py apply <decision_path> --account paper_default --confirm
```

## 6. 常见阻塞原因

```text
PRICE_MISSING
PRICE_TIME_MISMATCH
DATA_BLOCKED
NON_TRADING_DAY
ACTION_NOT_TRADABLE
LOW_CONFIDENCE_RECORD_ONLY
NO_BUY_BUDGET
CASH_NOT_ENOUGH_FOR_ONE_LOT
SINGLE_POSITION_LIMIT
DAILY_ROLLOVER_MISSING
DUPLICATE_DECISION_EXECUTION
```

如果出现 `NON_TRADING_DAY`，说明当前决策只能做研究或计划，不能生成可执行买入计划。

如果出现 `DAILY_ROLLOVER_MISSING`，先执行：

```powershell
python .\paper_trading.py rollover --account paper_default --trade-date <交易日>
```
